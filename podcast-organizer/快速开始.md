# 🚀 快速开始示例

## 使用方式

### 方式1: 直接告诉AI
```
帮我整理播客稿 /path/to/file.md
```

### 方式2: 描述性请求
```
整理这个播客采访，保留所有对话
```

### 方式3: 带选项请求
```
整理播客稿到 /Users/mumu/Downloads/trae/obsidian/
```

## AI会自动执行

1. 📁 **创建文件夹**
   ```
   /Users/mumu/Downloads/trae/obsidian/播客/
   或
   /Users/mumu/Downloads/trae/obsidian/采访/
   ```

2. 📖 **读取文件**
   - 分析文件类型
   - 识别说话者
   - 提取关键信息

3. ✨ **整理内容**
   - 提取核心主题
   - 编号对话Part
   - 提取金句
   - 整理资源引用

4. 📄 **生成文档**
   - Obsidian格式
   - 完整对话保留
   - 增值信息添加

5. ✅ **确认完成**
   - 显示文件位置
   - 显示文件信息
   - 提供预览
