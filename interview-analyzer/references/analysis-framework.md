# 采访提问分析框架

## 一、提问类型分析

### 1.1 按问题结构分类

#### 开放式问题 (Open-ended)
**特征**：无法用简单"是/否"回答，鼓励详细阐述
**识别标志**：以"什么"、"如何"、"为什么"、"谈谈"开头
**示例**："你是怎么走上这条路的？"
**分析要点**：观察是否给予受访者足够表达空间

#### 封闭式问题 (Closed-ended)
**特征**：可以用简短回答（是/否/数字/选择）
**识别标志**：以"是否"、"有没有"、"多少"、"哪个"开头
**示例**："你当时紧张吗？"
**分析要点**：用于确认事实，但过度使用会限制深度

#### 假设性问题 (Hypothetical)
**特征**：设定虚拟情境，探索可能性和价值观
**识别标志**：包含"如果"、"假设"、"设想"
**示例**："如果重来一次，你会怎么做？"
**分析要点**：测试受访者的反思能力和价值观

### 1.2 按提问目的分类

#### 探索式问题 (Exploratory)
**功能**：开拓新话题，挖掘未知信息
**特征**：话题转换、引入新维度
**示例**："当时还有什么细节让你印象深刻？"

#### 确认式问题 (Confirmatory)
**功能**：核实信息准确性，确保理解正确
**特征**：重复、总结、澄清
**示例**："所以你的意思是...对吗？"

#### 对比式问题 (Comparative)
**功能**：通过对比揭示深层含义
**特征**：涉及"和...相比"、"过去vs现在"
**示例**："这和之前有什么不同？"

### 1.3 按信息层次分类

| 层次 | 目的 | 典型问法 |
|------|------|----------|
| 事实层 | 获取客观信息 | "什么时候发生的？" |
| 行为层 | 了解具体行动 | "你是怎么做的？" |
| 认知层 | 探索想法和判断 | "你是怎么看的？" |
| 情感层 | 挖掘情绪体验 | "当时你感觉怎样？" |
| 价值层 | 触及核心信念 | "这对你意味着什么？" |

## 二、提问节奏分析

### 2.1 时间维度

#### 提问密度 (Question Density)
- **高密度**：短时间内连续提问，不给受访者喘息空间
- **中密度**：平衡的节奏，有适当的停顿
- **低密度**：大量留白，让受访者主导节奏

#### 间隔时机 (Timing)
- **即时追问**：在受访者话音刚落时追问
- **停顿后追问**：刻意等待2-3秒后再提问
- **话题转换时机**：在受访者说完一个完整观点后转换

### 2.2 节奏模式

#### 漏斗式节奏
```
[宽泛] → [具体] → [细节]
例如：行业趋势 → 公司策略 → 具体决策
```

#### 波浪式节奏
```
[深入] → [拉远] → [再深入]
例如：个人经历 → 行业背景 → 深层反思
```

#### 螺旋式节奏
```
[话题A] → [话题B] → [回到A但更深]
例如：失败经历 → 学习方法 → 对失败的重新定义
```

### 2.3 节奏控制技巧

| 技巧 | 表现 | 效果 |
|------|------|------|
| 停顿 (Pause) | 提问后保持沉默 | 给受访者思考空间，可能引出意外内容 |
| 加速 (Accelerate) | 快速连续提问 | 制造紧迫感，测试反应能力 |
| 减速 (Decelerate) | 放慢语速，拉长提问 | 营造深度交流氛围 |
| 重复 (Repeat) | 重复受访者关键词 | 鼓励继续展开 |

## 三、引导技巧分析

### 3.1 追问技巧 (Probing)

#### 垂直追问 (Vertical Probing)
**目的**：在同一话题上深入挖掘
**技巧**：
- "能具体说说吗？"
- "那个时刻发生了什么？"
- "你是怎么做出这个决定的？"

#### 水平追问 (Horizontal Probing)
**目的**：在同一层次上扩展广度
**技巧**：
- "还有其他类似的例子吗？"
- "这在其他方面有什么体现？"
- "团队里其他人怎么看？"

#### 反问追问 (Reflective Probing)
**目的**：通过反问让受访者自我反思
**技巧**：
- "你为什么这么觉得？"
- "这对你意味着什么？"
- "你怎么看待这个想法？"

### 3.2 话题转换技巧 (Transition)

#### 桥梁式转换
**特征**：利用受访者提到的关键词平滑过渡
**示例**：受访者提到"压力" → "说到压力，当时是如何处理的？"

#### 对比式转换
**特征**：通过对比引入新话题
**示例**："既然谈到成功，那我们聊聊失败的教训？"

#### 递进式转换
**特征**：逻辑上的层层递进
**示例**：个人经历 → 团队影响 → 行业意义

### 3.3 拉回技巧 (Redirecting)

#### 温和拉回
**话术**："这是个很有趣的观点，我们稍后再深入，先回到刚才说的..."

#### 明确边界
**话术**："为了更好地理解你的核心观点，我想聚焦在..."

#### 总结确认
**话术**："所以你的主要观点是...我理解对了吗？"

### 3.4 高级引导技巧

#### 投射技巧 (Projection)
**目的**：让受访者从旁观者视角看自己
**示例**："如果你朋友遇到这种情况，你会给他什么建议？"

#### 角色置换 (Role Reversal)
**目的**：换位思考，揭示盲点
**示例**："如果你是当时的决策者，你会怎么做？"

#### 极端情境 (Extreme Scenario)
**目的**：测试价值观边界
**示例**："如果必须在A和B之间选择，没有第三种可能？"

## 四、情绪引导分析

### 4.1 情绪识别维度

#### 显性情绪 (Explicit Emotion)
- 直接表达的情绪词汇："我很愤怒"、"那时候特别开心"
- 强烈的语气词和重复
- 长时间的停顿或哽咽

#### 隐性情绪 (Implicit Emotion)
- 语速变化（加快=兴奋/紧张，变慢=沉思/低落）
- 用词选择（防御性词汇vs开放性词汇）
- 话题回避（突然转移话题可能触碰到敏感点）

### 4.2 情绪引导策略

#### 情绪营造 (Emotion Elicitation)

**营造安全感**：
- 开场白建立信任："这里所说的一切都可以保留隐私"
- 自我暴露："我也有过类似的经历"
- 非评判态度："无论你说什么，我都不会评判"

**激发情绪**：
- 细节唤醒："当时是什么颜色？什么声音？"
- 时间锚定："那一刻，你脑子里闪过的第一个念头是什么？"
- 身体感知："你身体有什么反应？"

#### 情绪调节 (Emotion Regulation)

**情绪升温**：
- 使用情绪化词汇："令人愤怒的是..."
- 强调冲突："这听起来和...很矛盾"
- 挑战假设："但你刚才说..."

**情绪降温**：
- 暂停：给受访者整理情绪的时间
- 总结："让我确认一下我理解了..."
- 正常化："很多人在这种情况下都会有这种感受"

### 4.3 情绪共鸣技巧

#### 镜像 (Mirroring)
**表现**：匹配受访者的语速、语气、用词风格
**效果**：建立潜意识层面的连接

#### 验证 (Validation)
**话术**：
- "这完全可以理解"
- "换作是我，可能也会有同样的感受"
- "这种感受很正常"

#### 命名 (Labeling)
**技巧**：帮助受访者识别和命名自己的情绪
**示例**："听起来你当时感到既兴奋又有些不安？"

### 4.4 情绪处理红线

**应该做**：
- 承认和接纳所有情绪表达
- 给受访者情绪反应的空间
- 在情绪高峰时保持陪伴

**不应该做**：
- 急于用理性分析打断情绪
- 否定或淡化情绪（"这没什么大不了的"）
- 在没有建立信任时深入情绪话题
